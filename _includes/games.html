Flappy Fest

Auteur : Florian Fries

Flappy Fest est la version laser du célèbre jeu Flappy Fest. Aidez votre smiley à survivre aux méchantes colonnes qui traversent l'écran ! Jeux pour 2 joueurs.

Code source : https://github.com/FriesFlorian/edge-laser-flappyfest

Edge Steroid

Auteur : Yannick Jost

Clone d'Asteroid. Vous combattez votre adversaire aux commandes de votre vaisseau spatial. Détruisez-le avec votre rayon laser, mais attention aux astéroïdes flottants dans l'espace, et ne vous laissez pas surprendre par l'absence de gravité !

Code source : https://github.com/yanjost/edge-laser-python
